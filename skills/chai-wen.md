# 拆文技能

拆文是系统化分析优秀小说作品的核心方法，通过拆解爆款故事的结构、情节、人物、节奏，掌握可复用的写作模板。

## 一、拆文准备

### 工具准备
- Excel表格（推荐）
- 文档表格
- 拆文模板（见下方）

### 拆文模板

```
篇名：
作者：
类型：
平台数据：阅读量/评论数/点赞数
总字数：

分段分析表：
| 段落 | 字数 | 故事核 | 场景/人物 | 情节 | 亮点/爽点 | 节奏/冲突/反转 |
```

## 二、六维拆解法

### 维度1：故事核提炼

**定义**：一句话概括故事的核心冲突与走向

**公式**：
```
【谁】+【在什么困境】+【做了什么】+【结果如何】
```

**案例——诡异的准考证**：
> 因爱生恨的追求者霸凌被追的女孩，最后自食恶果的故事

**案例——媚鱼**：
> 皇上对我宠爱有加，白月光回来后冷落我，我看清虚情假意后反击复仇

**练习**：用50字以内提炼任意一个故事的故事核

---

### 维度2：三幕结构拆解

#### 经典三幕剧

| 幕 | 功能 | 核心问题 |
|---|------|----------|
| 第一幕 | 打破完美世界 | 主角面临什么困境？ |
| 第二幕 | 主角做出改变 | 如何应对？障碍是什么？ |
| 第三幕 | 重回完美世界 | 如何解决？结局如何？ |

#### 起承转合结构（中国式）

| 阶段 | 内容 | 案例《媚鱼》 |
|------|------|-------------|
| **起** | 角色现状+故事开端 | 白月光回来，皇上开始冷落我 |
| **承** | 矛盾冲突+主角行动 | 被欺辱、看清楚渣男、反击 |
| **转** | 意外+转折 | 小病猫被杀、复仇开始 |
| **合** | 解决问题+实现目标 | 渣男贱女都死、女主死去 |

#### 悬疑/规则类故事结构

以《诡异的准考证》为例：

| 阶段 | 内容 | 核心冲突 |
|------|------|----------|
| 第一幕 | 规则危机出现，主角孤立无援 | 纸条规则降临 |
| 第二幕上 | 出现同盟，找到破局方法 | 同伴合作 |
| 第二幕下 | 发现幕后黑手 | 真相揭露 |
| 第三幕 | 发现自己才是罪魁祸首 | 惊天反转 |

### 维度3：情节拆解

#### 情节点提取方法

1. **按场景切分**：每个场景=一个情节点
2. **记录四要素**：
   - 这个场景发生了什么？
   - 主角的目标是什么？
   - 遇到了什么障碍？
   - 结果如何？

#### 案例——《媚鱼》情节发展

```
开端：
1. 白月光回来，皇上冷落我
2. 白月光找我兴师问罪，皇上打我的婢女
3. 看到皇上腰间挂着白月光的荷包，心凉

发展：
4. 披风被扯坏，我把宋明嫣推进水里
5. 皇上要我送披风赔罪，我不同意晕倒，得知病入膏肓
6. 婢女帮我救助小病猫，重新燃起希望
7. 李宗格来看我，心又热乎起来
8. 宋明嫣因小病猫摔倒，李宗格弃我而去
9. 宋明嫣要定情信物，我扔地上让他捡
10. 宫宴上李宗格对我关心，我觉得虚情假意骂了他
11. 李宗格把我的兰花田换成牡丹，拆掉秋千

高潮：
12. 李宗格为气我让宋明嫣带走小病猫
13. 我哭求留下小病猫，宋明嫣添油加醋
14. 小病猫被带走——希望被扼杀
15. 小病猫拖着断腿爬来找我死去
16. 我用马鞭抽打宋明嫣，吐血晕倒
17. 李宗格后悔祈求原谅，我不感动
18. 我将计就计表演恋旧，李宗格言听计从
19. 我趁李宗格忙时抓宋明嫣到野猫岗，被野猫咬死

结局：
20. 生辰之日数落伤害，摇头痛哭乞求原谅
21. 我死去，爹爹带着骨灰回乡
22. 皇城被破，皇上死在我寝宫，手拿荷包和戒子
```

### 维度4：人物弧光拆解

#### 人物分析四要素

| 要素 | 问题 | 案例《媚鱼》女主 |
|------|------|-----------------|
| 人设 | 什么身份/背景/性格？ | 镇国候之女，嚣张跋扈，长得像白月光 |
| 目标 | 想要什么？ | 为小病猫报仇，渣男贱女都不放过 |
| 行动 | 做了什么？ | 把宋明嫣推水里、带去野猫岗杀掉 |
| 结果 | 结局如何？ | 病入膏肓死去 |

#### 主配角对照表

| 角色 | 人设 | 目标 | 主要行动 | 结果 |
|------|------|------|----------|------|
| 女主周媚鱼 | 像白月光、嚣张跋扈、被宠爱 | 复仇 | 反击、杀宋明嫣 | 死去 |
| 男主李宗恪 | 自私虚伪、装深情 | 享齐人之福 | 冷落我、维护宋明嫣 | 死在我寝宫 |
| 宋明嫣 | 白月光、倍受宠爱 | 抢走女主一切 | 杀小病猫 | 被野猫咬死 |

### 维度5：爽点/虐点拆解

#### 常见爽点类型

| 类型 | 说明 | 案例 |
|------|------|------|
| 复仇爽 | 坏人得到惩罚 | 宋明嫣被野猫咬死 |
| 打脸爽 | 当众羞辱坏人 | 把戒子扔地上让宋明嫣自己捡 |
| 逆袭爽 | 从弱变强 | 女主从被欺负到掌控全局 |
| 反差爽 | 前后对比强烈 | 皇上从冷落到言听计从 |

#### 常见虐点类型

| 类型 | 说明 | 案例 |
|------|------|------|
| 失去虐 | 失去重要的人/物 | 小病猫被杀 |
| 误解虐 | 被误解无法解释 | 被白月光诬陷 |
| 牺牲虐 | 为他人牺牲 | 女主死去 |
| 亲情刀 | 家人受伤害 | 爹爹辞官带骨灰回乡 |

#### 猫猫刀技法

《媚鱼》中的"小病猫"是经典的**情感寄托物**写法：

1. **建立羁绊**：小病猫是女主在绝望中的希望
2. **摧毁羁绊**：宋明嫣杀死小病猫
3. **爆发情绪**：女主的愤怒达到顶点
4. **复仇动机**：为小病猫报仇是复仇的正义性来源

### 维度6：节奏与悬念拆解

#### 节奏控制——节拍器法

每个场景是一个**节拍**，主角与障碍的博弈要层层推进：

```
场景1：冲突升级 → 场景2：更大冲突 → 场景3：极限对峙
```

《诡异的准考证》节奏示例：

| 段落 | 字数 | 节奏效果 |
|------|------|----------|
| 62字 | 纸条规则出现 | 快速建立悬念 |
| 254字 | 前桌死亡 | 第一波冲突 |
| 515字 | 新的规则去救校霸 | 情节拉丝 |
| 469字 | 校霸也死了 | 反转再反转 |

#### 制造紧张氛围的方法

1. **增加障碍**：目标越难实现越紧张
2. **时间限制**：deadline倒计时
3. **危机逼近**：危险时刻逼近
4. **情绪递进**：用情绪为剧情加码

#### 悬念技巧

| 技巧 | 说明 | 案例 |
|------|------|------|
| 问题带领 | 用问题引导读者 | "凶手是谁？" |
| 关键时刻停顿 | 在高潮时切断 | 考试结束前破案 |
| 草蛇灰线 | 埋下伏笔later回收 | 准考证是破案关键 |

## 三、复盘与借鉴

### 借鉴清单

1. **情节设计**：哪些情节让你"没想到"？
2. **情绪节奏**：在哪里哭了/爽了/紧张了？
3. **人物塑造**：配角是否鲜活？
4. **悬念设置**：哪些地方让你想追下去？
5. **开头结尾**：如何开头/收尾？

### 拆文检验问题

- [ ] 故事核是否清晰？（50字内可概括）
- [ ] 三幕结构是否完整？（起承转合/三幕式）
- [ ] 人物弧光是否完整？（目标→行动→结果）
- [ ] 爽点/虐点是否有层次？
- [ ] 节奏是否紧凑？（无冗余情节）
- [ ] 悬念是否有效？（能吸引追读）

### 可复用的写作模板

#### 追妻火葬场模板

```
1. 开端：男主冷落女主
2. 发展：女主被欺负、看清楚男主
3. 高潮：女主反击/复仇
4. 结局：男主追悔莫及/女主离去
```

#### 规则类悬疑模板

```
1. 规则降临：主角陷入神秘规则
2. 寻找规律：尝试理解规则
3. 同伴出现：找到合作者
4. 真相揭露：发现幕后黑手
5. 最终对决：解决危机
```

## 四、拆文输出格式

完成一篇小说的拆解后，输出以下内容：

```markdown
# 《XX》拆文报告

## 基本信息
- 篇名：
- 类型：
- 字数：
- 数据：

## 故事核
（50字内）

## 结构拆解
### 三幕/起承转合
### 情节点列表

## 人物分析
### 主角
### 配角

## 亮点与爽点

## 写作借鉴

## 疑问与思考
```

## 五、实战练习

### 练习1：提炼故事核
阅读一篇小说，用50字内提炼故事核

### 练习2：三幕结构图
用思维导图拆解小说的三幕结构

### 练习3：爽点定位
标注文中所有让你觉得"爽"的情节点

### 练习4：仿写练习
选择一篇喜欢的拆文，用相同结构写自己的故事

---

**核心原则**：拆文不是为了模仿，而是为了理解故事的本质规律，将规律内化为自己的写作能力。
