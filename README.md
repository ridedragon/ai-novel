# AI 小说写作助手

这是一个基于 React + Vite + Tailwind CSS 开发的 AI 小说写作前端网页。

## 功能特点

- **API 配置**：支持自定义 OpenAI 格式的 API Key、Base URL 和模型名称。
- **创作模式**：输入大纲或要求，AI 自动生成小说内容。
- **实时流式响应**：生成内容实时打字机效果展示。
- **Markdown 渲染**：支持格式化的文本展示。
- **响应式设计**：适配不同屏幕尺寸。

## 运行方式

项目已配置为运行在本地端口 **8002**。

1. 安装依赖（如果尚未安装）：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 打开浏览器访问：
   http://localhost:8002

## 使用指南

1. 点击右上角的 **设置** 图标。
2. 输入你的 API Key（例如 OpenAI 或其他兼容服务的 Key）。
3. 如果使用第三方服务，修改 Base URL。
4. 输入 Prompt 开始创作。
