🚀 使用指南

1. 查看浏览器日志

  1. 打开主应用 http://localhost:8002
  2. 按 F12 打开开发者工具
  3. 切换到 "Console" 标签
  4. 尝试生成正文内容
  5. 观察日志输出，特别是包含 "[Stream]" 的行

2. 运行测试脚本

  1. 在项目根目录打开终端
  2. 运行 node test-streaming-complete.js
  3. 这将模拟完整的流式输出流程
  4. 查看模拟的 UI 更新过程
