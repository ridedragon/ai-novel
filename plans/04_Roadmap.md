# AI 小说生成器项目计划书 - 第四部分：功能规划与迭代路线图

## 4.1 核心功能全景图（Feature Map）

本项目的功能规划围绕“创作全生命周期”展开，涵盖从脑洞萌发到最终校对的全流程：

### 4.1.1 灵感与策划阶段

* **Agent 脑洞碰撞**：用户提供一个关键词，多个 Agent（如“恐怖风专家”、“治愈系专家”）分别给出故事梗概并互相点评。
* **智能冲突生成器**：根据当前角色设定，自动设计“不可能调和的矛盾”，驱动剧情发展。
* **多维大纲树**：支持多级嵌套大纲，每一级都可由 Agent 自动细化或手动调整。

### 4.1.2 设定维护阶段

* **实时角色轨迹记录**：Agent 自动记录角色在每一章的地理位置变化、身体状态（如伤病）、情感关系变化。
* **设定自洽性检查**：每当用户手动修改设定时，Agent 自动检查是否与已发布的章节冲突。
* **世界观自动扩充**：当剧情涉及未定义的文明或设定时，Agent 根据风格基调自动“打补丁”。

### 4.1.3 自动执行阶段

* **全自动无人驾驶（Auto-Pilot）模式**：只需设定终点，Agent 自动完成从大纲到正文的所有中间步骤。
* **断点续写与多分支尝试**：针对同一个剧情点，Agent 同时生成 3 个不同走向的版本供用户选择。
* **风格锁定与迁移**：确保全书文风高度统一，支持“文风指纹”提取。

### 4.1.4 质检与发布阶段

* **全书逻辑热力图**：通过可视化图表显示全书逻辑严密程度，标注疑似“吃书”的段落。
* **读者视角模拟器**：Agent 模拟不同类型的读者（如“老书虫”、“逻辑怪”、“情感党”）对章节进行打分和反馈。
* **一键排版与出版级校对**：自动处理标点符号、段落缩进及敏感词过滤。

## 4.2 迭代路线图（Development Roadmap）

我们将项目分为四个关键里程碑：

### Milestone 1：Agent 核心大脑（短期）

* **目标**：实现“导演 Agent”和“创作 Agent”的基础协同。
* **关键任务**：
  * 构建 Agent 间通讯协议（JSON Schema）。
  * 在 UI 界面增加 Agent 任务队列显示。
  * 实现基础的“创作-审计”闭环。

### Milestone 2：深度记忆系统（中期）

* **目标**：解决 10 万字以上的一致性问题。
* **关键任务**：
  * 集成向量数据库（RAG）。
  * 实现分层递归摘要机制。
  * 开发角色/世界观的 Graph-RAG 图谱可视化组件。

### Milestone 3：全自动写作流（长期）

* **目标**：实现从脑洞到初稿的端到端自动化。
* **关键任务**：
  * 开发多分支剧情推演功能。
  * 完善“逻辑热力图”质检系统。
  * 支持用户自定义 Agent 插件。

### Milestone 4：生态与协同（远景）

* **目标**：建立创作者与 Agent 的共生社区。
* **关键任务**：
  * 建立 Agent 市场（分享 Prompt 预设、性格模组）。
  * 支持多人协作创作模式（多用户 + 多 Agent）。

## 4.3 商业化探索（可选规划）

1. **SaaS 订阅模式**：针对专业网文作家提供高性能 Agent 服务。
2. **API 开放平台**：将小说的“逻辑一致性质检”能力开放给其他写作平台。
3. **IP 孵化辅助**：利用 Agent 自动生成人物小传、世界观百科、甚至改编剧本。

---
*注：本路线图根据当前技术趋势制定，将根据实际开发进展进行滚动更新。*
